# ExpressTemplate

## Description
Template of an Express Project. 

language : nodeJS
framework : express 
view : ejs
css : sass

## How this was made?
`mkdir Express`
`cd Express`
`npm init`
`npm install -g express`
`npm install -g express-generator`
`express --view=ejs --css=sass --git ExpressTemplate`
`cd ExpressTemplate`
`npm install`
`npm start`
